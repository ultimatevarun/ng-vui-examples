Getting Started

Get up and running with VUI in minutes

1

Install

npm install @vui/accordion
2

Import

import { VuiAccordion } from '@vui/accordion';
3

Use

<ng-vui-accordion title="Hello"></ng-vui-accordion>

Prerequisites

Required Dependencies

  • Angular 15+
  • TypeScript 4.8+
  • Tailwind CSS 3.0+
  • RxJS 7.5+

Recommended Setup

  • Angular CLI latest
  • Node.js 16+
  • NPM 8+
  • VS Code with Angular extensions
